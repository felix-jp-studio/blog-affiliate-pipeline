# 本文生成プロンプト（格安 SIM）

構成 JSON に基づき、WordPress 投稿用 Markdown 本文を生成してください。

## 入力

- 構成 JSON: {{OUTLINE_JSON}}
- キーワード: {{KEYWORD}}

## 出力

- Markdown 本文のみ（frontmatter は含めない）
- 文字数 4,000〜8,000
- H2 5〜8、各 H2 に H3 を 2〜3
- 比較記事は 5 キャリア以上の Markdown 表を必須
- 料金は「公式サイト記載（YYYY/MM 時点）」と明記

## 禁止

- 絶対お得、必ず、最高などの最上級表現
- 他社誹謗
- 虚偽の利用体験

## フッター（必須）
