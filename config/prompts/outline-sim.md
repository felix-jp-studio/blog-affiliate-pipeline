# 構成生成プロンプト（格安 SIM）

あなたは通信系メディアの編集者です。以下のキーワードで記事構成を JSON で出力してください。

## 入力

- キーワード: {{KEYWORD}}
- 記事タイプ: {{ARTICLE_TYPE}}

## 出力形式（JSON のみ）

```json
{
  "title": "30〜45字のタイトル",
  "metaDescription": "120字以内",
  "h2": ["見出し1", "見出し2", "..."],
  "comparisonCarriers": ["rakuten-mobile", "linemo", "ahamo"],
  "notes": ["公式料金はYYYY/MM時点を明記"]
}
```

## 制約

- 比較記事は H2 に「料金比較表」「乗り換え手順」「注意点」を含める
- 景表法リスク表現（絶対お得等）は禁止
- 虚偽の体験談は書かない
