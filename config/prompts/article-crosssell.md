# 本文生成プロンプト（固定費×通信クロスセル）

{{SYSTEM_COMMON}}

## 入力

- 構成 JSON: {{OUTLINE_JSON}}
- キーワード: {{KEYWORD}}

## 出力

- Markdown 本文のみ（frontmatter は含めない）
- 文字数 3,500〜5,500
- H2 5〜7、各 H2 に H3 を 1〜3
- **2段階 CV 構成**（固定費セット → 通信キャリア公式）

## 必須セクション

1. **結論サマリ** — 固定費×通信をまとめて見直すメリット（断定せず「要公式確認」）
2. **固定費セットの比較** — Markdown 比較表（電気・セット割・対象回線）
3. **通信回線との組み合わせ** — セット割の前提条件、対象外ケース
4. **第1 CV（固定費）** — 電気・セットサービスの公式確認を促す（料金は要公式確認）
5. **第2 CV（通信）** — 既存キャリア（楽天モバイル / au / ソフトバンク光 等）への導線。アフィリエイト URL は本文に直接書かず、プレースホルダ `{AFFILIATE:carrier-id}` を使用
6. **向いている人・向いていない人**
7. **FAQ**（3 問以上）

## アフィリエイト

- 通信キャリアは `config/affiliate-rules.json` の carrier ID を `{AFFILIATE:rakuten-mobile}` 形式で挿入
- 電気サービスは **公式 URL のみ**（ASP 未登録の場合はプレースホルダを使わない）
- 最低 2 リンク（通信キャリア）を本文に含める

## 禁止

- セット割の金額を断定
- 虚偽の利用体験
- 「絶対お得」「必ず儲かる」等

## フッター（必須）

> 本記事は AI 支援により作成されています。料金・キャンペーン・セット割条件は変更される場合があります。申し込み前に各公式サイトで最新情報をご確認ください。
