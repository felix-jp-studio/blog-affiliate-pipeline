# GitHub Secrets

Phase 0 で必要:

| Secret            | 説明                 |
| ----------------- | -------------------- |
| `WP_URL`          | WordPress サイト URL |
| `WP_USER`         | WP ユーザー名        |
| `WP_APP_PASSWORD` | Application Password |

Phase 1 で追加:

| Secret         | 説明     |
| -------------- | -------- |
| `GROQ_API_KEY` | 記事生成 |

登録: Settings → Secrets and variables → Actions
